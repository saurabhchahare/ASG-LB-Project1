USAGE

This project includes small helper scripts to setup, build, run, and test the repository.

PowerShell (Windows):
- `scripts\setup.ps1` — prepares environment (install deps, create folders)
- `scripts\build.ps1` — builds the project (placeholder)
- `scripts\run.ps1` — runs the main program (placeholder)
- `scripts\test.ps1` — runs tests (placeholder)

Bash (cross-platform):
- `scripts/run.sh`, `scripts/build.sh`, `scripts/test.sh`

Usage examples:

PowerShell (run in project root):

```powershell
./scripts/setup.ps1
./scripts/build.ps1
./scripts/run.ps1
./scripts/test.ps1
```

Bash (Linux/macOS or Git for Windows):

```bash
bash ./scripts/build.sh
bash ./scripts/run.sh
bash ./scripts/test.sh
```

Each script is a small, documented placeholder — replace commands with real build/test steps.
